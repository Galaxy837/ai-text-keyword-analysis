## 项目简介
基于Python+Flask的轻量AI文本分析工具，实现两大核心功能：
1.  **关键词提取**：基于词频统计，可自定义提取数量、词长过滤、停用词
2.  **情绪分类**：简单规则分类，支持积极/消极/中性三分类
3.  前端Web界面，固定显示学号姓名，支持参数调优、跨平台迁移

## 可调节参数
- `TOP_K`：提取关键词数量（默认5，可修改）
- `MIN_WORD_LEN`：最短词长度过滤（默认2，可修改）
- `STOP_WORDS`：停用词表，可自定义优化
- 情绪分类词表，可自定义扩展

## 运行方法
1.  安装依赖：`pip install -r requirements.txt`
2.  启动服务：`python app.py`
3.  访问：`http://127.0.0.1:5000`
